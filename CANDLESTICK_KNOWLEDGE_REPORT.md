## Appendix: SQL Validation Queries

### Count events and outcomes per candle
```sql
SELECT c.candle_name, c.category, COUNT(e.id) as event_count, COUNT(o.id) as outcome_count
FROM candlestick_concepts c
LEFT JOIN candlestick_events e ON c.candle_name = e.candle_name
LEFT JOIN candlestick_outcomes o ON e.id = o.event_id
GROUP BY c.candle_name, c.category
ORDER BY event_count DESC;
```

### Check for NULL forward returns
```sql
SELECT candle_name, ticker, timestamp, return_1, return_3
FROM candlestick_outcomes
WHERE return_1 IS NULL OR return_3 IS NULL
ORDER BY candle_name, ticker, timestamp;
```

### Verify no duplicate events
```sql
SELECT ticker, timestamp, timeframe, candle_name, COUNT(*) as count
FROM candlestick_events
GROUP BY ticker, timestamp, timeframe, candle_name
HAVING COUNT(*) > 1;
```
